# Portfolio Website (Static)

A clean, responsive, one-page portfolio built with plain **HTML + CSS + JavaScript**.

## Run locally

Option A (fastest):
- Open `index.html` in your browser.

Option B (recommended):
- Use the VS Code **Live Server** extension, or
- Run a local server (Python):
  - `python -m http.server 8000`
  - Visit `http://localhost:8000`

## Customize

Edit `index.html` and replace:
- Your name, role, city
- Email + GitHub + LinkedIn links
- Projects (title, description, tech tags, live/code links)
- Experience entries

Swap the avatar:
- Replace `assets/img/avatar.svg` with a real image (same filename), or
- Update the `<img>` in the hero section to point to your image.

## Deploy (GitHub Pages)

1. Push this folder to a GitHub repository.
2. In GitHub: **Settings → Pages**.
3. Select **Deploy from a branch** and choose `main` (root).
4. Your site will be available at the provided Pages URL.
